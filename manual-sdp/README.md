# WebRTC 手動SDP交換版

完全サーバーレスのWebRTC接続アプリ。SDPテキストを手動でコピペして接続します。

## 特徴

- サーバー・BaaS不要
- 単一HTMLファイルで完結
- ローカルファイルとして開くだけで動作

## 使い方

### 接続を開始する側（Offer側）

1. `index.html` をブラウザで開く
2. 「カメラを許可する」をクリック
3. 「Offerを作成」をクリック
4. 表示されたSDPテキストをコピーしてLINE等で相手に送信

### 接続を受ける側（Answer側）

1. `index.html` をブラウザで開く
2. 「カメラを許可する」をクリック
3. 受け取ったSDPテキストを「相手のSDP」欄に貼り付け
4. 「Answerを作成」をクリック
5. 表示されたSDPテキストをコピーして相手に返信

### 接続完了

- Offer側は受け取ったAnswerを貼り付けて「Answerを適用」
- ICE候補も同様に交換して接続完了

## 注意

- HTTPS環境またはlocalhostが必要（カメラアクセスのため）
- 大きなテキストのコピペが必要なためUXは劣る
